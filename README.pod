=head1 XuLA FPGA Designs

Common and board-specific design files for the XuLA boards.

=over 4

=item * XuLA_lib

A directory containing HDL files describing modules that are useful in a variety of larger designs.

=item * 50

A directory containing design projects for the XuLA-50 board
(done using Xilinx ISE WebPACK 13).

=item * 200

A directory containing design projects for the XuLA-200 board
(done using Xilinx ISE WebPACK 13).

=back
